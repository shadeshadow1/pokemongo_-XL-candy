XL 사탕 입수 방법

XL 사탕은 아래와 같은 방법으로 입수할 수 있습니다.
*조건: 트레이너 레벨 40 이상*

  1. 일반 사탕 변환

    2. 일반 사탕 100개를 XL 사탕 1개로 변환할 수 있습니다.
    환전이라고 보시면 됩니다.

  2. 알 부화

    3. 2km 알 > 0 ~ 8개
    5km 알 > 0 ~ 12개
    7km 알 > 0 ~ 12개
    10km 알 > 0 ~ 12개
    12km 알 > 0 ~ 12개

  3. 포켓몬 포획

    4. 한 마리당 0 ~ 3개

  4. 포켓몬 교환

    5. 교환 1회당 0 ~ 1개

  5. 체육관 방어 중인 포켓몬에게 나무열매 주기

    6. 0.05% 확률로 1개 지급

  6. 전설의 포켓몬 포획

    7. 한 마리당 3 ~ 6개

  7. 박사에게 포켓몬 보내기

    8. 한 마리당 0 ~ 1개

  8. 파트너 포켓몬과 걷기

    9. 0~1개

  9. 이상한사탕 XL 사용

    10. 1개 사용시 1개 변환 (n개 사용시 n개 변환)
